# SoftDeck 시작 가이드

키보드 넘패드를 나만의 컨트롤러로 만들어 주는 앱입니다.
음악 재생, 볼륨 조절, 앱 실행, 웹사이트 열기 등을 넘패드 한 키로 할 수 있습니다.

---

## 1. 처음 실행하면

앱을 실행하면 화면 위쪽에 작은 검은색 창이 나타납니다.

```
┌─ SoftDeck ──────────────────┐
│  [폴더 트리]  │  [버튼 9개 그리드] │
└──────────────────────────────────┘
```

기본으로 미디어 컨트롤, 시스템 모니터, 앱 바로가기 등이 세팅되어 있어서
별도 설정 없이 바로 사용할 수 있습니다.

---

## 2. 넘패드로 조작하기

**Num Lock을 끄세요.** 그러면 넘패드가 버튼 리모컨이 됩니다.

```
키보드 넘패드         →    화면 버튼 위치

 [7] [8] [9]               [1행]
 [4] [5] [6]               [2행]
 [1] [2] [3]               [3행]

 [0] = 뒤로가기 (상위 폴더로 이동)
```

예를 들어, 기본 설정에서:
- **Num 7** → 볼륨 다운
- **Num 8** → 음소거
- **Num 9** → 볼륨 업
- **Num 5** → 음악 재생/일시정지

**Num Lock을 켜면** 넘패드는 원래대로 숫자 입력용으로 돌아가고, 창도 자동으로 숨겨집니다.

## 3. 기본으로 들어있는 버튼

### 메인 화면

| Num 7 | Num 8 | Num 9 |
|-------|-------|-------|
| 볼륨 - | 음소거 | 볼륨 + |
| **Num 4** | **Num 5** | **Num 6** |
| 이전 곡 | 재생/정지 | 다음 곡 |
| **Num 1** | **Num 2** | **Num 3** |
| CPU 사용률 | RAM 사용률 | Apps 폴더로 이동 |

### Apps 폴더 (Num 3으로 진입)

| Num 7 | Num 8 | Num 9 |
|-------|-------|-------|
| 계산기 | 메모장 | 탐색기 |
| **Num 4** | **Num 5** | **Num 6** |
| 작업관리자 | 화면캡처 | 명령프롬프트 |
| **Num 1** | **Num 2** | **Num 3** |
| Shortcuts 폴더 | YouTube | Google |

### Shortcuts 폴더 (Apps에서 Num 1로 진입)

| Num 7 | Num 8 | Num 9 |
|-------|-------|-------|
| 복사 | 붙여넣기 | 되돌리기 |
| **Num 4** | **Num 5** | **Num 6** |
| 잘라내기 | 전체선택 | 다시실행 |
| **Num 1** | **Num 2** | **Num 3** |
| 저장 | 스크린샷 | PC 잠금 |

> **Num 0**을 누르면 이전 폴더로 돌아갑니다.

## 4. 내 버튼 만들기

버튼을 **마우스 우클릭** → **Edit Button**을 누르면 편집 창이 열립니다.

### 할 수 있는 것들

| 하고 싶은 것 | 액션 타입 선택 | 예시 |
|-------------|---------------|------|
| 프로그램 실행 | Launch App | `notepad.exe`, `calc.exe` |
| 단축키 보내기 | Hotkey | `ctrl+c`, `alt+f4` |
| 텍스트 자동 입력 | Text Input | 자주 쓰는 문장, 이메일 주소 |
| 음악/볼륨 조절 | Media Control | 재생, 정지, 볼륨 등 |
| CPU/RAM 확인 | System Monitor | (자동 업데이트) |
| 다른 폴더로 이동 | Navigate Folder | 폴더 선택 |
| 웹사이트 열기 | Open URL | `https://youtube.com` |
| 여러 동작 연속 실행 | Macro | 핫키 → 대기 → 텍스트 입력 등 |

### 버튼 꾸미기

- **Label** — 버튼에 표시할 이름
- **Icon** — 아이콘 이미지 (png, jpg, ico 등). 아이콘 위에 글씨가 표시됩니다
- **Label Color** — 글씨 색상 변경 (색상 버튼 클릭 또는 `#ff0000` 같은 색상코드 입력)
- **Font Size** — 글씨 크기 변경 (기본 15px)

### 버튼 복사/붙여넣기

같은 설정의 버튼을 여러 개 만들고 싶을 때:
1. 원본 버튼 우클릭 → **Copy Button**
2. 빈 버튼 우클릭 → **Paste Button**

## 5. 폴더로 정리하기

버튼이 9개로 부족하다면 폴더를 만들어서 용도별로 나눌 수 있습니다.

### 폴더 만들기

1. 왼쪽 폴더 트리에서 **우클릭** → **New Sub-Folder**
2. 폴더 이름 입력
3. 새 폴더를 클릭하면 빈 버튼 9개가 나타남 → 원하는 대로 설정

### 폴더 이동 버튼 만들기

1. 버튼 우클릭 → Edit Button
2. 액션 타입을 **Navigate Folder**로 선택
3. 이동할 폴더 선택 → OK

> 폴더 안에 폴더를 만들 수 있어서, 원하는 만큼 버튼을 늘릴 수 있습니다.

### 자동 폴더 전환

특정 프로그램을 사용할 때 자동으로 해당 폴더로 바뀌게 할 수 있습니다.

1. 폴더 트리에서 폴더 우클릭 → **Edit (Mapped Apps)**
2. 연결할 프로그램 이름 추가 (예: `chrome.exe`, `photoshop.exe`)
3. 해당 프로그램이 포커스되면 자동으로 폴더가 전환됩니다

## 6. 창 다루기

### 이동/크기 조절
- 상단 바를 **드래그**하면 창이 이동합니다
- 창 **가장자리를 드래그**하면 크기가 바뀝니다
- 위치는 자동 저장되어 다음 실행 시 같은 위치에 나타납니다

### 숨기기/보이기
- **Num Lock ON** → 창 숨김 + 넘패드 숫자 입력 모드
- **Num Lock OFF** → 창 표시 + 넘패드 버튼 조작 모드
- **Ctrl + \`** (백틱) → 언제든 창 표시/숨기기 토글

### 투명도
- 상단 바의 슬라이더로 창 투명도를 조절할 수 있습니다 (20%~100%)

### 트레이
- 상단 바 **▼ 버튼** 또는 창 닫기 → 트레이로 최소화 (종료 아님)
- 트레이 아이콘 **더블클릭** → 창 다시 표시
- 트레이 아이콘 **우클릭** → Show / Settings / Reset Position / Quit

> 앱을 완전히 종료하려면 트레이 아이콘 우클릭 → **Quit**

---

## 7. 설정 변경

상단 바 **우클릭** → **Settings**

| 설정 | 설명 |
|------|------|
| Button Size | 버튼 크기 (기본 100px) |
| Button Spacing | 버튼 간격 (기본 8px) |
| Auto-switch | 활성 프로그램에 따라 폴더 자동 전환 |
| Always on top | 다른 창 위에 항상 표시 |
| Opacity | 창 투명도 |

---

## 8. 설정 백업/복원

내가 만든 버튼 설정을 파일로 저장하거나, 다른 PC에 옮길 수 있습니다.

- 상단 바 우클릭 → **Export Config** → JSON 파일로 저장
- 상단 바 우클릭 → **Import Config** → JSON 파일에서 불러오기

---

## 9. 게임할 때

| 게임 화면 모드 | 넘패드 조작 | 창 보임 |
|---------------|------------|---------|
| 보더리스 (창모드) | O | O |
| 전체화면 | O | X |

대부분의 게임은 보더리스 모드를 사용하므로 정상 동작합니다.
버튼을 클릭해도 게임 포커스가 빠지지 않습니다.

## 요약

| 키 | 동작 |
|----|------|
| **Num 1~9** (Num Lock OFF) | 버튼 실행 |
| **Num 0** | 상위 폴더로 이동 |
| **Num Lock 토글** | 버튼 모드 / 숫자 모드 전환 + 창 표시/숨김 |
| **Ctrl + \`** | 창 표시/숨기기 |
| **버튼 우클릭** | 편집 / 복사 / 붙여넣기 / 초기화 |
| **상단 바 우클릭** | 설정 / 내보내기 / 가져오기 |
| **트레이 우클릭** | 표시 / 설정 / 위치초기화 / 종료 |
