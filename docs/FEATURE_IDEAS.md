# SoftDeck 기능 아이디어

현재 기능 기반으로 정리한 추가 기능 후보 목록.

---

## 바로 쓸모 있는 것들

### 토글 버튼
- [ ] 두 상태를 번갈아 실행 (예: 마이크 음소거 ON/OFF)
- [ ] 버튼 색상/아이콘이 상태에 따라 변경
- [ ] `ActionConfig`에 `toggle_action` 추가하거나, 새 `toggle` 액션 타입으로 구현

### 오디오 장치 전환
- [ ] 스피커 ↔ 헤드폰 원클릭 전환
- [ ] pycaw으로 기본 재생 장치 변경 가능
- [ ] 현재 활성 장치를 버튼 라벨에 표시

### 버튼 딜레이 실행 (Hold)
- [ ] 짧게 누르면 A, 길게 누르면 B
- [ ] 실수 방지용: 잠금/종료 같은 위험한 동작에 유용
- [ ] `DeckButton`에 press duration 감지 추가

### 폴더별 그리드 크기
#### 시작전 단축키는 어떻게 할 것 인지
- [ ] 메인은 3x3, 특정 폴더는 4x5 같이 폴더마다 다른 레이아웃
- [ ] `FolderConfig`에 `grid_rows`/`grid_cols` 옵션 추가

## 편의성 개선

### 버튼 다중 선택 + 일괄 편집/이동/삭제
- [ ] Ctrl+클릭으로 여러 버튼 선택
- [ ] 선택한 버튼을 한번에 삭제, 다른 폴더로 이동, 스타일 일괄 변경

### 검색
- [ ] 전체 폴더에서 버튼 이름/액션 타입/파라미터로 검색
- [ ] 검색 결과 클릭 시 해당 폴더로 이동 + 버튼 하이라이트

### 폴더/버튼 Import/Export
- [ ] 전체 config가 아니라 개별 폴더 단위로 공유
- [ ] 폴더 우클릭 → Export Folder / Import Folder
- [ ] 커뮤니티 프리셋 공유에 유용

### Undo/Redo
- [ ] 버튼 편집/삭제/이동 실수 복구
- [ ] config 변경 히스토리 스택 (최근 N개)
- [ ] Ctrl+Z / Ctrl+Shift+Z
